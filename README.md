FIXORA: Real-Time Cinema Explorer
FIXORA is a dynamic web application that allows users to browse, search, and explore a vast database of films in real time. By leveraging professional-grade APIs, the app provides up-to-date information on the latest blockbusters, trending titles, and timeless classics.

 Features
Live Search: Find movies instantly as you type using asynchronous API fetching.

Dynamic UI: A responsive grid layout that adapts to mobile, tablet, and desktop screens.

Detailed Insights: View movie posters, ratings, release dates, and plot summaries.

Real-Time Updates: Data is pulled directly from the API, ensuring users always see current information.

 Built With
HTML5: Structured semantic markup for accessibility and SEO.

CSS3: Custom styling featuring Flexbox/Grid and smooth hover transitions.

JavaScript (ES6+): Utilizes the Fetch API for asynchronous data handling and DOM manipulation.

 Installation & Setup
Clone the repository:

Bash
git clone https://github.com/emmanuelouma16160/Movie-App-em-visko.git
API Key Configuration:

Sign up for an API key at TMDB or OMDb.

Open script.js and replace the API_KEY placeholder with your unique key.

Launch:

Open index.html in your preferred browser.

 How it Works
The application uses the Fetch API to send requests to the movie database endpoints. Once the JSON data is returned, JavaScript dynamically generates HTML cards to display the results.
